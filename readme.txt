**Evenza – Multi College Event Registration Portal (Project Description)**

Evenza is a scalable and beginner-friendly web-based application designed to manage and streamline event registration across multiple colleges. The system is built using Core PHP and MySQL, following a structured and modular architecture for easy maintenance and future scalability.

The project is divided into well-organized directories such as configuration files, reusable includes, authentication modules, and role-based dashboards for Admin, Organizer, and Student. Each user role has specific functionalities, ensuring proper access control and a secure environment.

The database is designed using relational principles with multiple interconnected tables such as colleges, users, events, registrations, and entry passes. It supports key features like multi-college management, event categorization, participant tracking, and unique pass generation for event entry.

Key features of the system include:

* Multi-college event management
* Role-based authentication (Admin, Organizer, Student)
* Event creation, editing, and deletion
* Student event registration and pass generation
* Organizer tools for participant management and pass verification
* Optimized database structure with indexing for performance

The frontend uses HTML, CSS, and Bootstrap to provide a responsive and user-friendly interface, while the backend ensures secure data handling and efficient processing.

Overall, Evenza serves as a complete full-stack project demonstrating practical implementation of web development concepts, database design, and system architecture.
